DSM RAG Project

Diagnostic and Statistical Manual of Mental Disorders (DSM) RAG System

Project Overview

After speaking with clinical professionals, I learned that while the Diagnostic and Statistical Manual of Mental Disorders (DSM) is a key reference in diagnosis, most practitioners don’t have the time to consult it in detail for every patient. This insight inspired me to build a Retrieval-Augmented Generation (RAG) system tailored to the DSM—designed to help clinicians, researchers, or students access relevant diagnostic criteria quickly and think critically about alternative possibilities, all while staying grounded in authoritative source material.

The system uses a multi-step agentic pipeline to generate precise, context-aware responses to mental health questions:

System Workflow

  1. Ingest the DSM into a ChromaDB, pre-processing it into clean, queryable chunks for fast retrieval.
  2. Accept a user question via a Flask interface.
  3. Search a separate ChromaDB that stores conversation history to retrieve any relevant past context.
  4. Simultaneously query the DSM chunks for the most relevant content to support the question.
  5. Combine relevant conversation history and DSM excerpts to generate a grounded, GPT-powered answer.
  6. Include a “Clear History” feature so users can start fresh at any point.

Important Notes

  • This system is strictly non-commercial and built for exploration and educational purposes.
  • All answers are generated using only two sources: the DSM and the conversation history.
  • The system does not invent medical advice—it only responds within the limits of the DSM’s content.

Tech Stack

  • Python for backend orchestration
  • LangChain for chaining retrieval and generation steps
  • OpenAI API for context-aware response generation
  • ChromaDB for vector-based semantic search (DSM + history)
  • Flask for an interactive front-end

This project demonstrates how retrieval-augmented architectures can be adapted to high-stakes domains—offering speed, accuracy, and a fresh lens for diagnostic reflection, while remaining fully grounded in expert-vetted literature.
